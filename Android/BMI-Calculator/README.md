# BMI Calulator

>Formula used: BMI = weight / (height * height) 

## BMI Categories:
- Under Weight = BMI <= 18.5

- Normal Weight = BMI between (18.5 - 24.9)

- Over Weight = BMI between (25 - 29.9)

- Obesity = BMI >=30
